# BeaconTrace

Locate devices offline using peer-to-peer Bluetooth. Completely cloudless.

## Overview
BeaconTrace is a privacy-first, open-source alternative to cloud-based tracking networks. It uses local Bluetooth Low Energy (BLE) signals to detect and estimate the range of paired devices, without ever sending data to the internet.

## Getting Started (Web)

This is the web landing page for BeaconTrace.

First, run the development server:
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
